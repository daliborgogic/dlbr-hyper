# dlbr-hyper

Custom [hyperterm](https://hyper.is/) theme.

![](http://i.imgur.com/rlcX4Ct.png)

## How to use

Add `dlbr-hyper` to `plugins` in `~/.hyperterm.js`.

```js
...
plugins: [
  'dlbr-hyper'
],
...
```

Q: What font is that?

A: [Hack](https://github.com/chrissimpkins/Hack) Typeface designed for source code
